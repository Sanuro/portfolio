Project Overview

My Role: Sketching, wireframing, researching, interface design, data visualization, visual design, prototyping, testing, iteration

As a personal project, I was in charge of my own work and had to come up with a plan to involve people who had more experience in both design and engineering to complete my project.
I started off with sketching and wireframing to create high-fidelity mockups to continue iteratively and eventually created an interactive prototype to be tested internally with users at Roku.

Video Demo

Technologies used: Javascript, Vue, Vuex, HTML, CSS, Visual Studio Code

  • Prototyped an alternate keyboard layout for TV screens for an On-Screen keyboard using the T9 keyboard layout.
  • Showcased the product at a Roku event with 80% of attendees giving positive feedback.
  • Documented the work for all the employees and made the prototype readily available.
