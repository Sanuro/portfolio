Project Overview

My Role: front-end design, data training, API Calls

This project was created through training a machine learning algorithm to understand dog breed pictures. This was made for a hackathon which won 3rd place within the event.

Video Demo

Technologies used:
XCode, Swift, iOS Charts, Computer Vision API, Core Data, API Calls, Core Data, Core ML.

  • Uses Computer Vision API to identify dog breed after upload / taking picture
  • Compare seached dog breed's quantitative information with the average's and other breeds in radar and line chart
  • Bookmark dog breeds to find them easier
